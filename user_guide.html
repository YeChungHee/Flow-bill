할인 문의 접수
대기 → wait

신청 페이지에 접속하여 어음 할인을 문의합니다.

신청 페이지 열기 →

이렇게 진행하세요

  1. 사업자번호를 입력합니다. (000-00-00000 형식)
    이전에 신청한 적이 있으면 수취사명이 자동으로 표시됩니다.
  2. 수취사명을 확인하거나 직접 입력합니다.
  3. 발행사 사업자번호를 입력합니다.
    이전 발행사 목록이 있으면 선택할 수 있습니다.
  4. 발행사명을 확인합니다. (사업자번호 입력 시 자동완성)
  5. 어음금액을 숫자로 입력합니다. (콤마 자동 적용)
  6. 만기일을 선택합니다.
  7. 입금예정일을 확인합니다. (기본값: 오늘)
    사용일수가 자동 계산됩니다. (공휴일 자동 반영)
  8. 할인 조회 요청 버튼을 클릭하면 접수 완료!
💡
접수가 완료되면 대시보드(내 신청현황)로 자동 이동됩니다. 관리자가 할인 조건을 전달할 때까지 잠시 기다려 주세요.

최초 신청 시 사업자번호로 계정이 자동 생성되므로 별도 등록 없이 대시보드에 바로 로그인할 수 있습니다. (초기 PIN: 000000)
⏳ 관리자가 문의를 확인하고 할인 조건을 전달합니다
2
할인 조건 확인 → 진행 또는 취소
선택 → select

관리자가 전달한 할인 조건을 확인하고 진행 여부를 결정합니다.

대시보드 열기 →

이렇게 진행하세요

  1. 대시보드에 접속하여 사업자번호 + PIN 번호로 로그인합니다.
    최초 로그인 시 초기 PIN(000000)을 입력하면 초기 설정 화면이 나타납니다.
    → 담당자 성명, 휴대폰 번호, 새 PIN 번호를 입력하세요.
  2. 상태가 선택인 건을 클릭합니다.
  3. 상세 모달에서 할인율, 수수료, 입금예정액을 확인합니다.
  4. 조건이 맞으면 진행요청 버튼을 클릭합니다.
    조건이 맞지 않으면 취소를 선택할 수 있습니다.
3
입금 계좌 입력 + 서류 제출
입력 → input

할인금을 받을 계좌 정보와 필요한 서류를 제출합니다.

이렇게 진행하세요

  1. 상태가 입력인 건의 상세 모달을 엽니다.
  2. 입금 계좌정보 (은행명, 계좌번호)를 입력합니다.
  3. 필요한 첨부파일 (어음 사본 등)을 업로드합니다.
    지원 형식: jpg, png, pdf / 원본 파일명으로 Google Drive에 자동 저장됩니다.
  4. 저장 버튼을 클릭하면 제출 완료!
💡
계좌정보(은행명·계좌번호)를 정확히 입력해 주세요. 입금 오류를 방지할 수 있습니다.
4
배서 완료 확인
배서 → endorse

어음 배서 대상 계좌를 확인한 뒤 배서 완료를 알립니다.

이렇게 진행하세요

  1. 상태가 배서인 건의 상세 모달을 엽니다.
  2. 배서 대상 계좌를 확인합니다. (분할배서인 경우 각 장별 금액도 확인)
  3. 실제 어음 배서를 진행합니다.
  4. 배서가 끝나면 배서완료 버튼을 클릭합니다.

할인내역서 확인 · 다운로드
완료 → done

입금이 완료되면 할인내역서를 확인하고 PDF로 저장할 수 있습니다.

이렇게 진행하세요

  1. 상태가 완료인 건의 상세 모달을 엽니다.
  2. 할인내역서 보기 버튼을 클릭합니다.
  3. 새 탭에서 할인내역서가 열리면 내용(발행사, 금액, 할인율, 수수료, 지급금액)을 확인합니다.
  4. 다운로드 버튼을 클릭하면 PDF로 저장됩니다.

자주 묻는 질문

회원가입이 필요한가요?
별도 회원가입은 필요 없습니다. 사업자번호 10자리와 PIN 번호 6자리만 입력하면 바로 이용할 수 있습니다.
초기 PIN 번호는 무엇인가요?
초기 PIN은 000000 (6자리)입니다. 최초 로그인 시 담당자 성명, 휴대폰 번호와 함께 새 PIN을 설정하는 초기 설정 화면이 자동으로 표시됩니다.
PIN 번호를 잊어버리면 어떻게 하나요?
관리자(276홀딩스, 02-785-7080)에게 연락하시면 PIN을 초기화해 드립니다.
이전에 신청한 정보를 다시 입력해야 하나요?
아닙니다. 사업자번호를 입력하면 이전 신청 내역에서 수취사명과 발행사 목록이 자동으로 표시됩니다. 반복 신청 시 입력 시간을 줄일 수 있습니다.
만기일이 공휴일이면 어떻게 되나요?
시스템이 자동으로 익영업일로 조정합니다. 2024~2027년 한국 공휴일이 내장되어 있어 별도 확인이 필요 없습니다.
대시보드 데이터는 실시간으로 업데이트되나요?
10초 간격으로 자동 갱신됩니다. 관리자가 할인 조건을 전달하면 잠시 후 대시보드에 반영됩니다.
다른 업체의 신청 내역도 볼 수 있나요?
아닙니다. 본인의 사업자번호로 신청한 건만 조회할 수 있습니다. 타인의 데이터에는 접근할 수 없습니다.
할인내역서는 어떻게 저장하나요?
완료 상태의 건에서 '할인내역서 보기'를 클릭한 뒤 '다운로드' 버튼을 누르면 브라우저 인쇄 기능을 통해 PDF로 저장됩니다.
