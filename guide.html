파트 A. 신청자 가이드

신청자

A1. 할인 문의 접수

신청자 페이지에 접속하면 할인 문의 탭이 활성화된 상태로 표시됩니다.

신청 페이지 바로가기 →

입력 항목

구분필드설명
수취사 정보사업자번호000-00-00000 형식, 이전 신청 시 수취사명 자동완성
수취사 정보수취사명자동완성 또는 직접 입력
어음 정보발행사 사업자번호이전 발행사 목록에서 선택 가능
어음 정보발행사명사업자번호 입력 시 자동완성
어음 정보어음금액 (원)숫자 입력 시 콤마 자동 포맷
어음 정보만기일날짜 선택
어음 정보입금예정일기본값: 오늘 날짜
자동 계산사용일수만기일과 입금예정일 기반 자동 산정 (공휴일 반영)
Tip. 사업자번호를 입력하면 이전 신청 내역에서 수취사명과 발행사 목록이 자동으로 표시됩니다. 반복 신청 시 입력 시간을 줄일 수 있습니다.

모든 항목 입력 후 할인 조회 요청 버튼을 클릭하면 접수가 완료되고 자동으로 대시보드로 이동합니다.

신청자

A2. 내 신청현황 (대시보드)

내 신청현황 탭 또는 신청 후 자동 이동으로 진입합니다.

대시보드 바로가기 →

로그인

  • 할인 문의 시 입력한 사업자번호 + 6자리 PIN으로 인증합니다.
  • 초기 PIN은 000000이며, 최초 로그인 시 새 PIN으로 변경해야 합니다.
  • 로그인 후 상단에 업체명과 사업자번호(000-00-00000 형식)가 표시됩니다.

대시보드 구성

  • 요약 카드: 전체 신청 / 진행중 / 취소 / 완료 건수
  • 신청 목록: 상태, 입금예정일(YY-MM-DD), 발행사명, 어음금액, 입금액, 할인율, 만기일(YY-MM-DD) 표시
  • 목록은 입금예정일 최신순으로 정렬됩니다.
  • 목록의 건을 클릭하면 상세 모달이 열립니다.

상태별 액션

상태신청자 액션

선택 (select)할인 조건 확인 → 진행요청 또는 취소 선택
입력 (input)입금 계좌정보 + 첨부파일 제출 → 저장 클릭
배서 (endorse)배서 대상 계좌 확인 → 배서완료 버튼 클릭
완료 (done)할인내역서 보기 버튼으로 할인내역서 확인 및 PDF 저장
신청자

A3. 할인내역서

입금 완료(done) 상태의 건을 상세 모달에서 열면 할인내역서 보기 버튼이 표시됩니다.

  • 클릭 시 새 탭에서 할인내역서가 열림
  • 할인 조건 (발행사, 액면금액, 할인율, 만기일, 수수료, 지급금액) 확인
  • 다운로드 버튼으로 PDF 저장 (브라우저 인쇄 기능 활용)
  • 저장 후 탭 자동 닫힘

파트 B. 관리자 가이드

관리자

B1. 관리자 로그인

admin.html 접속 시 6자리 PIN을 입력하여 인증합니다.

관리자 페이지 바로가기 →

  • 관리자 URL은 비공개로 유지 (A: URL 보안)
  • PIN 입력 필요 (B: PIN 보안)
  • 두 가지 보안이 동시에 적용되는 A+B 보안 방식
관리자

B2. 관리자 대시보드

로그인 후 모든 신청 건이 상태별로 표시됩니다. 리스트는 입금예정일 최신순으로 정렬되며, 날짜는 YY-MM-DD 형식으로 표시됩니다.

상태별 관리자 액션

상태관리자 액션상세 설명
대기 (wait) 할인 조건 입력 → 전송 월할인율 입력 → 할인액/입금액 자동 계산. 분할배서 설정 가능 (1~5장). 확인 팝업 후 전송.
선택 (select) 신청자 응답 대기 신청자가 조건을 확인하고 진행/취소를 선택할 때까지 대기
입력 (input) 신청자 계좌 입력 대기 신청자가 계좌정보와 첨부파일을 제출할 때까지 대기
배서 (endorse) 배서 완료 확인 대기 신청자가 배서를 완료할 때까지 대기
배서완료 입금일 입력 → 입금완료 버튼 실제 입금 후 입금일을 입력하고 완료 처리. 상태가 done으로 변경됨.
관리자

B3. 할인 조건 입력 상세

대기(wait) 상태의 건을 클릭하면 할인 조건을 입력할 수 있습니다.

입력 항목

항목설명
월 할인율 (%)수수료율 입력 → 할인액과 입금액 자동 계산
분할배서미선택 / 2~5장 선택. 선택 시 각 장별 금액 입력 (합계 = 어음금액 일치 필수)

계산식

사용일수 = 만기일(익영업일) − 입금예정일 + 1
※ 사용일수가 30일 미만인 경우 최소 30일 기준으로 계산됩니다.

할인수수료 = 어음금액 × (월할인율 / 100) / 30 × 사용일수 + 1,500원

분할배서 시: 추가 장수 × 1,500원 가산
(예: 3장 → 2장 × 1,500 = 3,000원 추가)

입금액 = 어음금액 − 할인수수료

조건 입력 후 확인 팝업에서 내용을 검토한 뒤 전송합니다.

관리자

B4. 입금 완료 처리

신청자가 배서를 완료한 건은 관리자 대시보드에 "배서완료" 상태로 표시됩니다.

  • 실제 입금을 진행한 후 상세 모달에서 입금일을 입력하고 입금완료 버튼을 클릭합니다.
  • 입금 완료 시 신청자는 대시보드에서 할인내역서를 확인할 수 있게 됩니다.
관리자

B5. 첨부파일 관리 및 공유

신청자가 업로드한 첨부파일(매출계산서 등)은 Google Drive에 자동 저장되며, 관리자 상세 모달에서 확인·공유할 수 있습니다.

파일 저장 방식

  • 신청자 업로드 시 파일이 Google Drive 공유 폴더에 자동 저장됩니다.
  • 파일명은 날짜_수취사_발행사_어음액.확장자 형식으로 자동 변환됩니다.
  • 지원 형식: jpg, png, pdf (최대 5MB)

관리자 상세 모달 — 첨부파일 영역

버튼기능
파일 보기새 탭에서 Google Drive 파일 열기
링크 복사파일 공유 링크를 클립보드에 복사
공유모바일: 카카오톡·슬랙 등 앱 선택 공유 / PC: 링크 복사
Drive 폴더 열기파일 URL이 없는 기존 데이터의 경우, Drive 폴더로 이동
Tip. 모바일에서 공유 버튼을 누르면 카카오톡, 슬랙, 문자 등 설치된 앱 목록이 표시되어 파일을 바로 전달할 수 있습니다.

  • 관리자: URL 비공개 + PIN 인증 이중 보안
